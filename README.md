## Description

基于nestjs的server

## 下载

```bash
$ npm install
```

## 执行

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## 测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## 创建一整个CRUD

nest g resource cats module
nest g resource(创建CRUD) cats(整个模块名称) module (src下的文件夹名称)

选择src -> REST API -> n(CRUD)
